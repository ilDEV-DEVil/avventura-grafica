# Cartella immagini personaggi

In questa cartella dovrebbero essere inserite le immagini dei personaggi per l'avventura grafica.

## Nomi file richiesti:
- character1.png - Immagine per il Personaggio 1
- character2.png - Immagine per il Personaggio 2

## Specifiche:
- Formato: PNG (consigliato) o JPG
- Dimensioni ideali: 120x120 pixel o proporzioni quadrate
- Sfondo trasparente (per PNG)

## Percorsi nel config:
- Personaggio 1: /images/character1.png
- Personaggio 2: /images/character2.png

Aggiungi qui le tue immagini dei personaggi personalizzate!
